## ✨ Features

- ☀️ Sun with emissive glow  
- 🌎 Earth with texture and rotation  
- 🔴 Mars  
- 🟠 Jupiter  
   
- 🌌 Star background  
- 🎮 WASD camera controls  

---

## ⚙️ Implementation

- Built using **A-Frame (WebVR framework)**
- Used **sphere primitives** for planets
- Applied **realistic textures**
- Implemented:
  - Orbit animation (parent entity rotation)
  - Self-rotation (planet spin)
- Added **lighting + emissive materials** for realism
